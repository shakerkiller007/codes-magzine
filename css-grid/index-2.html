What is HTML?

HTML stands for H yper Markup Language. It's a markup languges that web develpors use to structure and describe the content of a webpage (not a programming language).

HTML consists of elements that describe different types of content: paragraph,links,headings,images. video,etc. web browsers understand HTML and render HTML code as websites.

In HTML each element is made up of 3 parts:

  • opening Tags
